# **App Name**: DZ Store Manager

## Core Features:

- Add New Purchase: Allows users to add details of a new purchase including supplier, items, and quantity.
- Add New Sale: Allows users to add details of a new sale including customer, items, and quantity.
- Manage Stock: Displays and allows editing of the current stock levels.
- Supplier Management: Manage suppliers and their contact information
- Customer Management: Manage customers and their information
- Automatic Stock Level Adjustment: Uses an LLM to check a supplier product list and adjusts current stock levels based on it.  This is a tool to streamline stock management.

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) for a professional and reliable feel.
- Background color: Very light indigo (#F0F8FF) for a clean and modern look.
- Accent color: Violet (#8A2BE2) for highlights and important actions.
- Body font: 'Inter', a sans-serif font, provides a clean, readable interface.
- Headline font: 'Space Grotesk', a sans-serif font, for a contemporary feel.
- Use a set of modern, minimalist icons from Shadcn/UI to ensure consistency and a clean user interface.
- Employ a card-based layout, similar to the provided image, but modernized using Shadcn/UI components. Focus on a clean, intuitive user experience.
- Subtle transition animations within Shadcn/UI components to improve the user experience.
